🌐 Deploy Guide
Step 1 — Create GitHub Account
Go to github.com → Sign up free → Create new repository named kodandroid
Step 2 — Upload Files
Upload index.html and admin.html to the repository
Step 3 — Enable GitHub Pages (FREE)
Settings → Pages → Source: main branch → Save
Your site: yourusername.github.io/kodandroid
Alternative — Netlify (Drag & Drop)
Go to netlify.com → Drop your folder → Instant live URL in 30 seconds! Completely FREE.
